This plug-in makes use of the following libraries/projects:

1. AWS Java SDK :: SDK Core. SDK Core runtime module holds the classes that are used by the individual service clients to interact with Amazon Web Services. Users need to depend on aws-java-sdk artifact for accessing individual client classes, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
2. AWS Java SDK :: Services :: Amazon S3. Amazon S3 module holds the client classes that are used for communicating with Amazon Simple Storage Service, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
3. AWS Java SDK :: Auth. Auth module holds the classes that are used for authentication with AWS services, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
4. AWS Java SDK :: Services :: AWS STS. AWS STS module holds the client classes that are used for communicating with AWS Security Token Service, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.

The library is released under Apache License, which can be found here https://aws.amazon.com/apache-2-0.

See the libs directory for more information.
